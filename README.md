Flutter Widgets Exploration Labs

This repository contains four Flutter lab assignments focused on mastering core UI widgets and application logic.
📂 Project List

    Profile Card Lab (profile_card_lab)

        Focus: Basic layouts using Scaffold, Column, Row, Container, and Card.

        Features: Profile image integration and text styling.

    Bottom Navigation Lab (bottom_nav_lab)

        Focus: Multi-screen navigation.

        Features: BottomNavigationBar implementation and state-driven screen switching.

    Catalog Lab (catalog_lab)

        Focus: Scrolling lists and grids.

        Features: Dynamic data rendering with ListView.builder and GridView.builder.

    Registration Lab (registration_lab)

        Focus: Forms and user input.

        Features: TextFormField validation, GlobalKey<FormState>, and success dialogs.# Flutter-App
